# Antigravity Skills

This repository contains a collection of skills designed for agents. These skills extend the capabilities of the agent, allowing for specialized tasks such as document generation, design, and internal communications.

## References

For more information on agent skills and standards, please refer to:
- [Anthropic Skills Repository](https://github.com/anthropics/skills)
- [Agent Skills Home](https://agentskills.io/home)

## Available Skills

- **algorithmic-art**: Generate art using algorithms.
- **atomic-commits**: Create atomic git commits. This agent skill was inspired by a prompt and concept shared by ArmaÄŸan Amcalar during his live stream. - **brand-guidelines**: Work with brand guidelines.
- **canvas-design**: Design implementation for canvas elements.
- **doc-coauthoring**: Collaborative document editing.
- **docx**: Manipulation and creation of Word documents.
- **frontend-design**: Frontend web design assistance.
- **internal-comms**: Internal communication tools.
- **mcp-builder**: Tools for building Model Context Protocol servers.
- **pdf**: PDF generation and manipulation.
- **pptx**: PowerPoint presentation generation.
- **skill-creator**: Meta-skill for creating new skills.
- **slack-gif-creator**: Create GIFs for Slack.
- **theme-factory**: Generate themes.
- **web-artifacts-builder**: Build web artifacts.
- **webapp-testing**: Testing tools for web applications.
- **xlsx**: Excel spreadsheet manipulation.
